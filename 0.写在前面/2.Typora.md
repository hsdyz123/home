# Typora

## 软件安装

- Typora 官网（需要翻墙）：https://support.typora.io/
- Typora 中文网：https://www.typora.net/
- Typora 下载地址：直接搜索即可。中文网上有详细的讲解，可自行查看

  > 注：Typora 在 `v1.0` 版本之后开始**收费**，最好下载 `v1.0` 之前的版本，`v0.9+`

## 主题修改

- 网址：https://www.typora.net/themes/
- 网址：https://theme.typoraio.cn/

> 注：可以直接下载，也可以自行修改 `css` 来实现

## 基本用法

> 注：用法右键都可以实现。快捷键在菜单栏都有。

标题

```markdown
段落内容直接写

# 一级标题（最大，适配文档大标题）

## 二级标题（常用，适配章节标题）

### 三级标题（常用，适配小节标题）

#### 四级标题

##### 五级标题

###### 六级标题（最小）

<!-- 文件目录 -->

[toc] / [TOC]
```

基础文本格式

```markdown
_斜体内容_

**加粗内容**

**_粗斜体内容_**

~~删除线内容~~

==高亮内容==（部分编辑器支持，如 VS Code）

> 引用内容
```

列表

```markdown
- 无序列表
  - 列表 1
  - 列表 2
  - 列表 3

1. 有序列表
   1. 列表 1
   2. 列表 2
   3. 列表 3

- [ ] 任务列表
- [x] 任务列表
```

链接与图片

```markdown
[链接显示文本](链接地址)
[GitHub 官网](https://github.com)

![图片加载失败提示语](图片地址)
![笔记封面](./images/cover.png)（相对路径）
![logo](https://static.char123.com/images/favicon.ico)
```

分割线

```markdown
---
```

表格

```markdown
| 表头 1 | 表头 2 | 表头 3 |
| ------ | ------ | ------ |
| 内容 1 | 内容 2 | 内容 3 |
| 内容 4 | 内容 5 | 内容 6 |

| 表头 1 |  表头 2  | 表头 3 |
| :----- | :------: | -----: |
| 左对齐 | 居中对齐 | 右对齐 |
| 内容 1 |  内容 2  | 内容 3 |
| 内容 4 |  内容 5  | 内容 6 |
```

脚注

```markdown
这是一段需要注释的文本[^1]
[^1]: 这里是脚注的内容（文档末尾显示）
```

代码块

- 单行：前后各一个单引号 `123`
- 多行：前后各三个单引号

```markdown
我是代码块内容
```

公式块

- 单行：前后各一个美元符号 $123$
- 多行：前后各两个美元符号

$$
我是公式块内容
$$

## 浏览器显示

在浏览器中下载 `Markdown Reader` / `Markdown Viewer` 插件添加至浏览器，即可在浏览器中查看 Markdown 文件

- `Markdown Reader`（自带目录）✅️
  - 插件网址：https://markdownreader.github.io/
- `Markdown Viewer`（不带目录）
  - 插件网址：https://github.com/simov/markdown-viewer
- 安装
  - 除谷歌浏览器外，其他浏览器可直接在插件网址下载安装，或者在各自插件商店搜索下载安装。安装完成后，点击插件详细信息，勾选允许访问文件 URL 后，即可在浏览器中查看 Markdown 文件，浏览器会自动转换成 HTML 文件
  - 谷歌浏览器可在下面网址下载插件。解压压缩包，打开扩展程序，打开开发者工具，将插件直接拖入即可安装。安装完成后，点击插件详情，勾选允许访问文件网址后，即可在浏览器中查看 Markdown 文件，浏览器会自动转换成 HTML 文件
    - 直接下载：<a href="./Markdown Reader 3.6.4.zip" download>Markdown Reader 3.6.4.zip</a>
    - Crx 搜搜：https://www.crxsoso.com/
    - 插件小屋：https://www.chajianxw.com/
- 使用
  - 安装完成后可点击右上角设置进行修改，建议打开深色模式，开启代码自动换行
  - 也可以自定义桌面样式

```css
h1 {
  text-align: center;
}
```
